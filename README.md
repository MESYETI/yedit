## Keys
| Key      | Actions             |
| -------- | --------------------|
| Ctrl + S | Save                |
| Ctrl + W | Save As             |
| Ctrl + O | Open                |
| Ctrl + T | Suspend to terminal |
| Ctrl + Q | Quit                |
| Ctrl + F | Find                |
